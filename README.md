# ⚽ FunToTo - 축구 점수 맞추기 토토

통신 AI 사업본부 구성원들과 축구 경기 점수를 예상하고 베팅하는 사내 토토 게임입니다.

## 시작 방법

### 1. Node.js 설치 확인

```bash
node --version
```

Node.js가 없다면 https://nodejs.org 에서 설치하세요.

### 2. 서버 실행

```bash
node server.js
```

### 3. 브라우저에서 접속

```
http://localhost:3000
```

> `index.html`을 파일로 직접 열면 데이터가 localStorage에 저장됩니다.  
> 서버를 통해 접속해야 `data.json` 파일에 저장됩니다.

---

## 주요 기능

- **경기 설정** — 홈팀 / 원정팀 이름 입력
- **점수 예상 베팅** — 이름, 예상 점수, 베팅 금액 입력
- **실시간 배당 계산** — 참여자 추가 시 배당률 및 예상 수령액 즉시 업데이트
- **수금 체크** — 참여자별 수금 여부 토글
- **결과 확정** — 실제 점수 입력 시 당첨자 및 수령액 표시
- **데이터 자동 저장** — `data.json`에 저장, 새로고침 후에도 유지
- **새 게임** — 헤더의 [새 게임] 버튼으로 초기화

## 관리자 / 관전자 모드

| 모드 | 설명 |
|------|------|
| 👁 관전자 (기본) | 점수 현황, 배당률 조회만 가능 |
| 🔑 관리자 | 참여자 등록, 수금 체크, 결과 입력 등 전체 기능 사용 가능 |

입력 필드 외부에서 **`QkrQkrdl`** 를 키보드로 입력하면 관리자 모드로 전환됩니다.  
다시 입력하거나 헤더의 [관리자 종료] 버튼을 누르면 관전자 모드로 돌아옵니다.
